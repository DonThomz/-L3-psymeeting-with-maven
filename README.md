### Executable

Dans le dossier shade lancer la commande :

```
java -jar L3-PsyMeeting.jar
```

### Build le projet

1. Pour build le projet :

2. importer le projet depuis votre IDE

3. (si besoin faire charger le ***pom.xml***)

4. lancer les tasks : 	

   ```
   javafx:run
   javafx:compile
   ```
